Overview

The BMIDE assistant is your guide to getting started with the BMIDE. The BMIDE is used to view and extend the Teamcenter data model and behaviors, deploy to a test Teamcenter environment for validation, go live with your extensions, and exit extensions in a live product environment.

All of these best practices have established process for you to follow. The links below provide more details on these best practices. Please take a moment to read and familiarize yourself with these processes.

The tabs to the left make it easy for you to follow these processes by showing you the BMIDE tool that is used for each step and linking you to additional information.
