Начало работы...


Добро пожаловать в Бизнес-разработчик IDE (BMIDE)! BMIDE используется для просмотра и работы с моделями данных в Teamcenter, запуска среды Teamcenter для проверки данных, обработки и изменения расширений в интерактивной рабочей среде. BMIDE поможет Вам справиться со всеми указанными задачами.

Приступая к работе, сперва требуется создать проект. Проект позволяет приложению BMIDE определить логику задачи, которую требуется решить, и выступает в роли контейнера для всех ваших расширений. Выберите любой из типов проекта, указанных ниже, чтобы просмотреть информацию о том, для чего он используется. Как только вы выбрали тип проекта, нажмите на кнопку Создать.

Создание нового шаблона BMIDE


Если вы первый раз работаете с расширениями в Teamcenter, то сначала требуется создать шаблон проекта. Шаблон выступает в роли контейнера для всех бизнес-объектов, свойств, а также логики, которые создаются в приложении BMIDE. Далее можно использовать данный шаблон, чтобы выполнить пробный запуск среды Teamcenter. После проверки можно передать данный шаблон на рабочий сайт для его использования другими пользователями.

Выполнение интерактивных обновлений существующего шаблона BMIDE


В ряде случаев может потребоваться внесение изменений в логику работы действующего сайта. Например, может появиться необходимость в добавлении значения из списка, статуса или контекста идентификатора. BMIDE использует данный тип проекта, чтобы помочь в выполнении интерактивных обновлений без выключения Teamcenter. Перед использованием данного типа проекта, должен быть создан и установлен шаблон проекта на рабочем сайте (смотрите выше).

Обновление шаблона BMIDE, созданного в предыдущей версии Teamcenter


Если вы выполняете обновление уже установленного решения Teamcenter до более новой версии, то сперва следует обновить шаблон BMIDE. Обновление вашего шаблона необходимо, поскольку предыдущая версия вашего шаблона будет несовместима с новой версией Teamcenter. Для обновления шаблона BMIDE, сперва установите новую версию клиента BMIDE, затем нажмите на значок "Обновить шаблон" и воспользуйтесь мастер-процессом, чтобы обновить текущий шаблон BMIDE до новой версии. Для получения информации о всех шагах процесса обновления, обращайтесь к "Руководству по обновлению".
