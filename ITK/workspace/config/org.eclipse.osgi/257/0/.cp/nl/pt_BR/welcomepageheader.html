Como Começar...


Bem vindo ao BMIDE! O BMIDE é usado para exibir e estender o modelo de dados do Teamcenter e seu comportamento, implantar um ambiente de teste do Teamcenter para validação, entrar no ar com as suas extensões e editá-las em um ambiente de produção ao vivo. O BMIDE lhe ajudará a realizar essas atividades.

Para começar você precisará criar um projeto. O projeto ajuda o BMIDE a entender qual o tipo de atividade você quer realizar e irá atuar como um contêiner para todas as suas extensões. Clique em qualquer um dos seguintes tipos de projeto para saber para quê eles são utilizados. Depois de identificar o tipo de projeto que irá criar, clique no botãoCriar para começar.

Criar um novo Modelo de BMIDE


Se esta é a primeira vez que você irá estender o Teamcenter, é necessário criar um projeto modelo. O modelo irá atuar como um contêiner para todos os objetos de negócio, propriedades e comportamentos que você criar dentro do BMIDE. Então você pode implantar esse modelo em um ambiente do Teamcenter para teste. Após a validação você pode instalar o modelo em um site de produção para que seus usuários o utilizem.

Execute Atualizações ao Vivo em um Modelo BMIDE existente


Ocasionalmente é possível editar o comportamento de um site em produção que está sendo executado ao vivo. Por exemplo, é possível adicionar um valor LOV, um status ou um contexto de identificação. O BMIDE utiliza este tipo de projeto para ajudá-lo a executar atualizações ao vivo sem derrubar o sistema do Teamcenter. Antes de utilizar este tipo de projeto, um Projeto Modelo deve ser criado e instalado no site de produção (veja acima).

Atualize o Modelo BMIDE de uma versão anterior do Teamcenter


Se você estiver no processo de atualização de instalação do Teamcenter para uma nova versão do Teamcenter, primeiro, é necessário atualizar o Modelo BMIDE. A atualização do seu modelo é necessária desde que a versão anterior do seu modelo não seja compatível com a nova versão do Teamcenter. Para atualizar o seu modelo BMIDE, primeiro instale a nova versão do cliente BMIDE e então clique no ícone "Atualizar Modelo" e utilize o assistente para atualizar seu modelo BMIDE atual para a nova versão. Para um conjunto completo de etapas de Atualização, consulte o "Guia de Atualização"
