Guía de iniciación...


!Bienvenidos a BMIDE! Se utiliza BMIDE para ver y extender el modelo de datos y los comportamientos de Teamcenter, implementar un entorno de prueba para la validación de Teamcenter, realizar las extensiones en vivo y editar las extensiones en un entorno de producción en vivo. BMIDE le ayudará a completar estas tareas.

Primeramente deberá crear un proyecto. El proyecto ayuda a BMIDE a entender la clase de actividad que usted desea implementar y BMIDE actuará como un contenedor de todas las extensiones. Pulse uno de los tipos de proyecto a continuación para ver para qué se usan. Una vez identificado el tipo de proyecto que desea crear, pulse el botón Crear para comenzar.

Creación de una plantilla de BMIDE nueva


Si esta es la primera vez que extiende Teamcenter, primeramente deberá crear un proyecto de plantilla. La plantilla actuará como un contenedor de todos los objetos de negocio, las propiedades y los comportamientos que cree en BMIDE. Seguidamente podrá implementar esta plantilla en un entorno de Teamcenter a fines de prueba. Después de la validación, podrá instalar la plantilla en un sitio de producción para que pueda ser utilizada por los usuarios.

Implementación de las actualizaciones en vivo en la plantilla de BMIDE


Puede que desee editar el comportamiento de un sitio de producción con una ejecución en vivo. Por ejemplo, se aconseja agregar un valor, estatus o un contexto de Id de la LOV. BMIDE utiliza este tipo de proyecto para ayudarle a realizar las actualizaciones en viso sin tener que apagar el sistema de Teamcenter. Antes de utilizar este tipo de proyecto, deberá haber creado un proyecto de plantilla y haberlo instalado en el sitio de producción (consulte la indicación anterior).

Actualización de la plantilla de BMIDE a partir de una versión anterior de Teamcenter


Si está por actualizar la instalación de Teamcenter con una versión nueva de Teamcenter, primeramente deberá actualizar la plantilla de BMIDE. Es necesario actualizar la plantilla porque la versión anterior de la plantilla no es compatible con la versión nueva de Teamcenter. Para actualizar la plantilla de BMIDE deberá instalar la versión nueva del cliente de BMIDE, seguidamente deberá pulsar el icono “Actualizar la plantilla” y utilizar el asistente para actualizar la plantilla actual de BMIDE con la versión nueva. Si desea obtener un conjunto completo de los pasos de actualización, consulte “Actualizar la guía”.
