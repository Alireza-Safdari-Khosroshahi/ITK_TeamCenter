Willkommen bei BMIDE. BMIDE wird zur Anzeige und Erweiterung des Teamcenter-Datenmodells und -Verhaltens, zur Bereitstellung für eine Teamcenter-Testumgebung für Validierungszwecke, zur Aktivierung Ihrer Erweiterungen sowie der Bearbeitung Ihrer Erweiterungen in einer Produktionsumgebung verwendet. Mit BMIDE können Sie alle diese Aktivitäten ausführen.

Zuerst erstellen Sie ein Projekt. Mit diesem Projekt werden Daten für BMIDE zur Verfügung gestellt, welche Aktivität Sie ausführen möchten, und das Projekt dient als Container für alle Ihre Erweiterungen. Klicken Sie nachfolgend auf einen beliebigen Projekttyp, um Informationen anzuzeigen, für welche Aktivitäten die verschiedenen Projekttypen verwendet werden. Nachdem Sie den Projekttyp ermittelt haben, den Sie erstellen möchten, klicken Sie auf die Schaltfläche Erstellen.

Erstellen einer neuen BMIDE-Vorlage


Wenn Sie zum ersten Mal eine Teamcenter-Erweiterung erstellen, müssen Sie zuerst ein Vorlagenprojekt erstellen. Die Vorlage dient als Container für alle Geschäftsobjekte, Eigenschaften und Verhalten, die Sie mit BMIDE erstellen. Danach können Sie diese Vorlage für eine Teamcenter-Umgebung zum Testen bereit stellen. Nach abgeschlossener Validierung können Sie die Vorlage für eine Produktions-Site installieren, so dass Ihre Benutzer die Vorlage verwenden können.

Ausführen von Live-Updates für eine vorhandene BMIDE-Vorlage


Es kann der Fall eintreten, dass Sie das Verhalten einer aktiven Produktions-Site bearbeiten möchten. Sie können beispielsweise einen Wertelistenwert, Status oder ID-Kontext hinzufügen. BMIDE verwendet diesen Projekttyp, um Sie bei der Ausführung von Live-Updates zu unterstützen, ohne die Teamcenter-Umgebung herunterfahren zu müssen. Bevor Sie diesen Projekttyp verwenden können, müssen Sie ein Vorlagenprojekt erstellt und für die Produktions-Site installiert haben (siehe oben).

Upgrade Ihrer BMIDE-Vorlage aus einer früheren Version von Teamcenter


Wenn Sie ein Upgrade Ihrer vorhandenen Teamcenter-Installation durchführen, müssen Sie zuerst Ihre BMIDE-Vorlage aktualisieren. Das Upgrade Ihrer Vorlage ist erforderlich, da die frühere Version Ihrer Vorlage nicht mit den neuen Versionen von Teamcenter kompatibel ist. Installieren Sie zur Aktualisierung Ihrer BMIDE-Vorlage zuerst die neue Version des BMIDE-Clients, klicken Sie danach auf das Symbol "Vorlage aktualisieren" und verwenden Sie den Assistenten zur Aktualisierung Ihrer vorhandenen BMIDE-Vorlage auf die neue Version. Informationen zu allen Arbeitsschritten zur Ausführung von Upgrades finden Sie im "Upgrade-Handbuch".
