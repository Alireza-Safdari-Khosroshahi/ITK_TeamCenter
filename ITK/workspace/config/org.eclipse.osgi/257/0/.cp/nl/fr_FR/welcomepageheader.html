Prise en main...


Bienvenue dans le BMIDE. Le BMIDE (Environnement de développement intégré du Modèle de travail) est utilisé pour afficher et étendre le modèle de données Teamcenter et les comportements, déployer vers un environnement de test pour validation, mettre en oeuvre vos extensions, et modifier des extensions en direct dans un environnement de production. Le BMIDE vous aidera à accomplir ces activités.

Pour commencer, vous devez créer un projet. Un projet informe le BMIDE sur le type d'activité que vous voulez réaliser, et il sert de conteneur pour toutes vos extensions. Cliquez sur les types de projet suivants pour savoir à quoi ils sont utilisés. Après avoir identifié le type de projet que vous voulez créer, cliquez sur le bouton Créer pour commencer.

Créer un nouveau modèle-type de BMIDE


Si c'est la première fois que vous réalisez une extension de Teamcenter, il faut d'abord créer un modèle-type de projet. Le modèle-type sert de conteneur pour tous les objets de gestion, les propriétés et les comportements que vous créez dans le BMIDE. Ensuite, vous pouvez déployer ce modèle-type vers un environnement Teamcenter pour le tester. Après validation, vous pouvez installer le modèle-type dans un site de production pour que vos utilisateurs l'exploitent.

Effectuer des mises à jour en direct sur un modèle-type BMIDE existant


Parfois, vous avez besoin de modifier le comportement d'un site de production en cours de fonctionnement. Par exemple, vous souhaitez ajouter une valeur LOV, un Statut ou un Contexte d'ID. Le BMIDE utilise ce type de projet pour vous aider à effectuer des mises à jour en direct sans arrêter votre système Teamcenter. Avant de pouvoir utiliser ce type de projet, vous devez avoir créé et installé un modèle-type de projet dans le site de production (voir ci-dessus).

Mettre à niveau votre modèle-type BMIDE depuis une version antérieure de Teamcenter


Si vous êtes en cours de mise à niveau de votre installation Teamcenter vers une nouvelle version, il faut d'abord mettre à niveau votre modèle-type BMIDE. La mise à niveau de votre modèle-type est nécessaire, car sa version précédente ne sera pas compatible avec la nouvelle version de Teamcenter. Pour mettre à niveau votre modèle-type BMIDE, installez d'abord la nouvelle version du client BMIDE, ensuite cliquez sur l'icône “Mettre à niveau le modèle-type” et utilisez l'assistant pour mettre à niveau votre modèle-type actuel à la nouvelle version. Pour connaître toutes les étapes de la mise à niveau, consultez “Upgrade Guide”.
